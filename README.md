# IFRS 9 Expected Credit Loss (ECL) Automation

Python and DuckDB SQL project demonstrating the automation of **IFRS 9 Expected Credit Loss (ECL)** calculations and financial data processing.

The project implements an end-to-end **ETL pipeline** for loading, transforming and analyzing financial data, including automated calculation of **PD, LGD, EAD**, and generation of the final ECL reporting dataset.

> **Note:** All source data, entity names and file paths have been anonymized for public release.

---

## Technology Stack

**Python • Pandas • NumPy • DuckDB SQL • IFRS 9 • ETL • Financial Data Analysis • Excel**
